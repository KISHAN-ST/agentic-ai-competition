# Agentic AI Competition

A small competition to get the hands-on experience with GitHub Copilot

## Task
- Fork this repository
- Clone your forked repository
- Use VSCode or Copilot Web to connect your repository
- Create a folder with your name
- Use Copilot to build a project. Use the agent mode to build it.
- Push the code to your repository and create a pull request here.

### Happy Coding!
